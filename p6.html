Project #6 Optical Illusion

Sketch by Yulia

Description

My idea was to create an interactive illusion in order to make the user's waiting time in front of the loading screen more interesting and engaging. For this project I was inspired by the bouncing DVD logo, Google T-Rex game and Winmap Music player animations. I used a simple “for loop” code to place the required number of circles on my drawn computer screen and then made it iterative with “variations”. I also added a function that changes colors when the mouse is clicked. I believe that such interactive elements will help kill the time when a waiting screen appears. Instead of experiencing the anger of waiting, the user can get distracted and play with the satisfying loop animation.

Reflection

I enjoyed working on it. The code itself is pretty simple and not that long. I made the computer and desktop image in Illustrator and then exported it as a png file. The most difficult part of this assignment for me was to adjust the desired relationship between the mouse movement and the radius changes of the circles. I spent most of my time studying and testing this part of the task.
